EDITANDO TAREFAS

A noção central do fluxo de trabalho no sistema é a tarefa. Um usuário abre uma tarefa para outro usuário quando necessitar que uma ou mais atividades sejam realizadas, em um dado prazo

É importante ressaltar que só é possível a criação de uma tarefa em um NUP quando existem assuntos e interessados.

Entrando no grid de Tarefas

1. No menu lateral esquerdo, clique em Tarefas> Administrativas:

2. O menu de tarefas é aberto e é possível realizar várias operações.

2. Na nova tela, clique ao lado da tarefa para expandir as opções existentes:

2. Na nova tela, clique ao lado da tarefa para expandir as opções existentes:

3. Clique em Editar Tarefa

4. O menu onde é possível modificar os parâmetros da tarefa é aberto. Após a modificação, aperte salvar.
