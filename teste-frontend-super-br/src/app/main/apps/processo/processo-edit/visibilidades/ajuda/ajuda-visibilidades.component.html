RESTRIÇÕES DE ACESSO

A configuração de restrição de acesso determina de que modo um processo/documento avulso pode ser acessado pelos usuários internos e externos do sistema.

Se não houver “Restrição de Acesso”, significa que todos os usuários internos (colaboradores) da Instituição podem acessar e visualizar o processo/documento avulso.

Caso a opção seja marcada por um usuário com perfil de coordenador, o processo/documento avulso passará a ter o acesso restrito.

A partir do momento em que um processo/documento avulso tenha acesso restrito, apenas o coordenador que criou a restrição tem acesso sobre ele, com poder “Master”.

Qualquer outro usuário terá acesso negado.

Deve-se então realizar a configuração da restrição de acesso, para liberar o acesso para usuários escolhidos pelo coordenador.

Apenas um usuário com poder “Master” pode liberar o acesso para outros usuários.

O acesso pode ser liberado para um usuário interno ou externo, ou para um setor, sendo que nesse último caso, todos os usuários lotados naquele setor estão automaticamente liberados.

Poderes que podem ser concedidos:

  • Master: o usuário com poder “Master” sobre um processo/documento avulso tem automaticamente os poderes de “Ver”, “Editar” e “Apagar”, mais o poder de atribuir poderes para outros usuários (inclusive o poder “Master”).
  • Ver: o poder de “Ver” o processo/documento avulso permite acessá-lo e visualizar seu conteúdo, mas não o editar.
  • Editar: o poder de “Editar” o processo/documento avulso permite editar seus dados.
  • Apagar: o poder de “Apagar” não se aplica aos processos/documentos avulsos, pois eles não podem ser apagados como regra geral.

APENAS o usuário com o Poder “Master” pode retirar poderes de outros usuários.

A liberação da restrição para um usuário externo específico, que já tenha se cadastrado, significa que o cidadão, ao consultar o sistema via internet, terá acesso mesmo se houver restrição de acesso. Um usuário externo não pode receber o poder “Master”.

Configurando uma Restrição de Acesso

1. No grid com opções de dados que podem/devem ser preenchidos, escolha o campo Restrições de Acessos

2. A página de restrições de acesso é aberta. É possível ver a lista de acessos e os poderes concedidos de cada usuário ou cadastrar uma nova permissão de acesso a partir do botão .

3. Apertando no botão Novo, a página de configuração da restrição de acesso é aberta. é possível escolher "Quem receberá acesso", sendo praticável dar acesso para um setor ou para um usuário. Clique na Lupa para escolher os usuário/setores possíveis.

4. Além de preencher quem receberá o acesso, também é escolhido o poder que vai ser concedido.

5. Clique em Salvar para finalizar o cadastramento de acesso.

6. A página de restrições é aberta novamente. Importante ressaltar que facilmente é possível retirar o acesso concedido, por meio da lixeira que fica na lateral direita de cada acesso.
