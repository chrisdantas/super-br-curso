CRIANDO PROTOCOLOS

FIQUE ATENTO:

Caso o processo esteja com alguma Tramitação Externa em andamento, a abertura da Tarefa não será realizada.

Em um NUP arquivado, só protocolo do gênero arquivístico podem ser criadas. Para liberação, solicite o desarquivamento por tarefa

ELIMINAR PROCESSO/DOCUMENTO AVULSO e RECOLHER PROCESSO/DOCUMENTO AVULSO AO ARQUIVO DEFINITIVO são protocolo de uso exclusivo dos Arquivistas.

MANTER SOB GUARDA TEMPORÁRIA NO ARQUIVO CORRENTE e RECOLHER PROCESSO/DOCUMENTO AVULSO AO ARQUIVO DEFINITIVO são protocolo que exigem a tramitação interna para o setor de Arquivo.

Um NUP anexado/apensado a outro NÃO permite a abertura de protocolo. Abra as protocolo no NUP principal.

Existem unidades que só podem receber as protocolo pelo seu Protocolo. Sendo assim, as protocolo NÃO podem ser mandadas diretamente para os usuários lotados em tal unidade.

Quando o NUP é restrito e o usuário NÃO possui poderes para visualizá-lo, ele NÃO consegue abrir uma tarefa. O acesso deve ser concedido antes da abertura da tarefa.

Apenas o usuário da tarefa pode modificá-la.

Em uma tarefa já concluída, NÃO é possível modificação.

Uma tarefa que já contém atividades NÃO pode ser excluída.

Caso NÃO seja informado um usuário responsável, o sistema vai utilizar a distribuição automática para a tarefa.
